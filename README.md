## Введение

**Advance Report** – это система для создания, обработки и управления авансовыми отчетами. Проект разработан с использованием монолитной архитектуры для серверной части и отдельного веб-приложения для пользовательского интерфейса.

Цель проекта – предоставить удобный инструмент для формирования отчетов, позволяющий пользователям:
- Создавать, редактировать и удалять отчёты.
- Просматривать и фильтровать отчёты.
- Получать аналитическую информацию на основе представленных данных.

---

## Особенности проекта

- **Монолитный backend:**  
  Основная бизнес-логика, обработка данных, формирование отчетов и взаимодействие с базой данных реализованы в едином приложении (папка `monolith` в репозитории).

- **Веб-интерфейс:**  
  Клиентская часть представлена в виде статического веб-приложения, размещённого на GitHub Pages ([https://ilimsky.github.io/advance_report/](https://ilimsky.github.io/advance_report/)). Интерфейс обеспечивает удобное взаимодействие с серверной частью через API.

- **API:**  
  Монолит предоставляет набор REST API для работы с отчетами, что позволяет как веб-приложению, так и сторонним системам интегрироваться с проектом.

---

## Архитектура

Проект состоит из двух основных компонентов:

1. **Монолитное приложение (Backend):**  
   - Реализует бизнес-логику, работу с базой данных и обработку запросов от клиента.
   - Расположено в каталоге `monolith`.
   - Обеспечивает набор REST API для операций с отчетами.

2. **Веб-приложение (Frontend):**  
   - Разработано для взаимодействия с пользователем.
   - Размещено на GitHub Pages по адресу [https://ilimsky.github.io/advance_report/](https://ilimsky.github.io/advance_report/).
   - Обращается к API монолитного приложения для выполнения CRUD-операций и получения аналитической информации.
